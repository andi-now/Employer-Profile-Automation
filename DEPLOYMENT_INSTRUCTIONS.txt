METHOD A: Drag & Drop (EASIEST)
---------------------------------

1. Go to: https://app.netlify.com/drop

2. Sign up/Login to Netlify (it's FREE!)

3. Drag and drop the ENTIRE "out" folder onto the page
   (The 'out' folder was created in Step 2)

4. Wait 30 seconds for deployment

5. Done! Your site is live at: https://random-name-12345.netlify.app


METHOD B: Netlify CLI (FASTER FOR FUTURE UPDATES)
---------------------------------------------------

1. Install Netlify CLI:
   npm install -g netlify-cli

2. Login to Netlify:
   netlify login
   (A browser window will open - login there)

3. Deploy:
   netlify deploy --prod --dir=out

4. Follow the prompts:
   - Create new site? Yes
   - Team: Select your team
   - Site name: employer-profile-automation

5. Done! Your URL will be shown in the terminal
